# Advent of Code 2024 (Rust)

My Advent of Code 2024 solutions, implemented in Rust.

## Project Layout

- `2024/`: single Rust crate (`advent_of_code_2024`)
- `2024/src/day_XX.rs`: one module per day (`01` through `25`)
- `2024/res/`: puzzle and sample input files
- `2024/src/lib.rs`: exports all day modules

## Verification

Run:

```bash
cd 2024
cargo test
```
